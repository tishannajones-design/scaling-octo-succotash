# RGX Engage Design System

The official design system for RGX Engage — governance consulting and platform services.

## Live Site

View the design system at: **https://tishannajones-design.github.io/rgx-design-system/**

## What's Included

- **Foundations** — Color system (primary, semantic, dark mode, contrast, accessibility), typography (9-level scale), 12-column grid, 8px spacing, elevation
- **30+ Components** — Buttons, Inputs, Select, Checkbox, Radio, Toggle, Badges, Avatar, Alerts, Toast, Cards, Modal, Table, Navigation, Tabs, Pagination, Progress, Stepper, Tooltip, Breadcrumb, Divider, Empty State, Skeleton
- **Patterns** — Form patterns, layout patterns (dashboard, detail view)
- **Resources** — Design tokens (JSON), WCAG 2.1 AA accessibility guide, developer implementation guide

## Design Tokens

Import `tokens.json` for use in any CSS-in-JS or design tool pipeline.

## Brand Architecture

| Service Line | Primary | Accent | Use |
|---|---|---|---|
| GaaS (Consulting) | Flow Blue `#0074e4` | Execution Orange `#f15a24` | Consulting services |
| GPaaS (Platform) | Intelligence Purple `#6c5ce7` | Growth Green `#00b859` | Platform products |

> Service color systems must never be mixed.
